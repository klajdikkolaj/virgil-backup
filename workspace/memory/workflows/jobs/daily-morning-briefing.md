# Job SOP - Daily Morning Briefing

Schedule intent: 07:00 Europe/Tirane daily.

## Objective
Produce a concise 2-minute morning briefing without using X/Twitter.

## Source policy
Use only publicly accessible sources:
- Official docs/changelogs/blogs (OpenClaw, vendors, GitHub releases/issues)
- Reliable tech news/public newsletters
- Hacker News/Product Hunt/GitHub trending when relevant
- arXiv or credible research summaries for AI + BCI topics
- Public LinkedIn posts only when directly accessible and verifiable

Reject login-only/paywalled/scraped content.

## Required sections (exact order)
1. Top Stories (3-5)
2. Interesting Threads (2-4)
3. OpenClaw & Agent Automation News
4. AI Breakthroughs & Mind-Digital Research
5. Action Items

## Output rules
- High-signal, no fluff
- Include links for each key item
- If a source lane is thin, say so briefly

## File output (mandatory)
Write full briefing to:
`/home/clawdkbot/virgil-vault/Daily/YYYY-MM-DD-briefing.md`

## Channel output
Post concise summary in the target Discord channel using same section structure.
